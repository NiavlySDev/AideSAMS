🏥 AideSAMS - Système de Documents Médicaux

Nouvelle fonctionnalité de gestion complète des documents médicaux

📋 Fonctionnalités implémentées

  • Documents remplissables directement sur le site (sans Google Docs)
  • Système de signature électronique intégré
  • Sauvegarde des signatures dans les paramètres
  • Export PDF et Word des documents
  • Partage sécurisé pour signature des parents (certificat de naissance)
  • Interface en mode blanc ressemblant aux documents originaux
  • Auto-sauvegarde des données saisies
  • Système de protection par mot de passe pour les partages

📄 Documents disponibles

🩺 Avis d'arrêt de travail

Document médical pour prescrire un arrêt de travail avec toutes les informations nécessaires.

👶 Certificat de naissance

Certificat officiel avec signature des parents via un système de partage sécurisé.

💰 Facture d'hospitalisation

Facture détaillée avec calcul automatique des totaux et export professionnel.

🔧 Instructions d'utilisation

  1. Accéder aux documents : Cliquez sur l'onglet "Documents" dans la navigation
  2. Sélectionner un type : Choisissez le type de document à créer
  3. Remplir le formulaire : Saisissez les informations requises
  4. Ajouter une signature : Cliquez sur "Signer" pour ajouter votre signature électronique
  5. Exporter : Utilisez les boutons "Export PDF" ou "Export Word"
  6. Partager (certificat) : Utilisez "Partager pour signature" pour les parents
